# Decision Ontology

Supported note types:

- decision-policy
- decision-frame
- tradeoff-rule
- hard-constraint
- soft-preference
- default-priority
- ask-trigger
- approval-rule
- decision-example
- counterexample
- rejection-pattern
- escalation-rule
- uncertainty-rule
- calibration-question
- wrong-decision
- corrected-decision
- context-scope
- meta-rule

Statuses: seed, tested, reliable, stale, retired, contradicted.

Confidence: low, medium, high.

Risk tiers: suggest-only, ask-before-action, reversible-auto, approval-required, never-auto.

Sensitivity: public, personal, private, secret. Secret items are rejected/redacted.
